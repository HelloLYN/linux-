查看支持的shell     cat /etc/shells
编写shell脚本
  1.#!/bin/bash
  2.赋予执行权限  chmod 755 hello.sh
  3.执行         ./hello.sh     bash/sh hello.sh

ctrl + u从光标所在位置删除到行首
!!重复执行上一条命令
set 查询系统中默认所有已经生效的变量,包括系统变量,自定义变量
set -u 使未定义的变量报错
把当前路径变量定义在全局环境变量中:PATH=$PATH:<path>
自定义环境变量export 变量名=变量值
子shell中只能读到环境变量,无法读到局部变量
env 查看环境变量

$?最后一次执行的命令状态.0->正确,非0->不正确执行
$$当前进程的进程号(pid)
$!后台进行的最后一个进程号(pid),类似于mongodb在后台运行
&后台进行

通配符
  ?匹配一个任意字符
  *匹配任意个字符
  []匹配括号里任意一个字符
  [^]匹配不是括号中的一个字符
